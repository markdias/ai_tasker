## How It Works: The Project Creation Flow

### Step 1: Initial Goal Input
You start by describing what you want to plan.

**Example:**
```
"I want to plan a birthday party"
```

### Step 2: AI Clarifying Questions
The app asks ChatGPT to generate 3-4 clarifying questions to better understand your project scope.

**Example Questions:**
- What is the date of the party?
- How many guests are expected?
- What type of party is it? (Birthday, Wedding, Corporate, etc.)
- What's your budget range?

### Step 3: Answer the Questions
You answer each question based on your preferences.

**Your Answers:**
- Date: November 15, 2025
- Guests: 20-30 people
- Type: Casual birthday gathering
- Budget: $300-500

### Step 4: Review & Create Project
Review your project details and create a project folder with a due date.

**Project Details:**
- Title: "Birthday Party - November 15"
- Description: "I want to plan a birthday party"
- Due Date: November 15, 2025

## Using Projects

### View All Projects
1. Tap the **Projects** tab at the bottom
2. See all your active and archived projects
3. Each project shows:
   - Project title and description
   - Progress percentage (tasks completed)
   - Number of completed vs. total tasks
   - Due date

### Create a New Project
1. Tap the **+** button in the Projects tab
2. Enter your goal description
3. Answer the AI-generated clarifying questions
4. Review and create the project
5. All tasks are automatically organized under the project

### View Project Details
1. Tap any project in the list
2. See:
   - All tasks for that project
   - Project information and due date
   - Progress toward completion
3. Edit project details or manage tasks

### Manage Tasks Within a Project
1. Open a project
2. Tasks are displayed in a list
3. You can:
   - ‚úì Mark tasks complete
   - üóëÔ∏è Delete individual tasks
   - üìù View task details (estimated time, priority)

### Edit a Project
1. Open the project
2. Change:
   - Project name/title
   - Description
   - Due date
3. Tap "Save Changes"

### Archive a Project
1. Open the project
2. Tap "Archive Project"
3. Project moves to "Archived Projects" section
4. Can be recovered or permanently deleted

### Delete a Project
1. Open the project
2. Tap "Delete Project"
3. All associated tasks are deleted
4. **Action is permanent**

## Best Practices

### 1. Be Specific with Goals
‚ùå Bad: "Plan stuff"
‚úÖ Good: "Plan a 50-person corporate team-building event in January"

### 2. Answer Questions Thoroughly
The more detailed your answers, the better the AI understands your needs.

### 3. Review Generated Tasks
- Read through all generated tasks
- Remove tasks that don't apply to your situation
- Add custom tasks as needed

### 4. Set Realistic Due Dates
- Consider the complexity of your project
- Account for time needed to complete tasks
- Build in buffer time

### 5. Update Progress Regularly
- Mark tasks complete as you finish them
- Helps you stay motivated
- Improves future AI suggestions

### 6. Archive When Complete
- Archive projects once finished
- Keeps your active project list clean
- Preserves history of past projects

## Troubleshooting

### "AI questions didn't make sense"
**Solution:** Your goal description may have been too vague. Try being more specific about what you're planning.

**Example:**
- ‚ùå "Plan a party"
- ‚úÖ "Plan a surprise 30th birthday party for 25 friends in December"

### "Generated tasks don't match my needs"
**Solution:** You can:
1. Edit individual tasks after creation
2. Delete tasks that don't apply
3. Add custom tasks in the project
4. Edit the project description and regenerate

### "Generated too many/too few tasks"
**Solution:** The number of tasks depends on your goal complexity. You can:
- Delete unwanted tasks after generation
- Add more specific details in your answers
- Manually add more tasks to the project

### "Due date keeps resetting"
**Solution:** Make sure you save your changes after editing the due date. The "Save Changes" button in the project detail view must be tapped.

## Tips for Success

### Tip 1: Start with Projects for Complex Goals
Use the Projects feature for anything with multiple related tasks. Use individual tasks for quick, standalone items.

### Tip 2: Name Projects Clearly
Good project names help you find things later:
- ‚úÖ "Birthday Party - November 15"
- ‚úÖ "Kitchen Renovation Q4 2025"
- ‚ùå "Project 1"
- ‚ùå "Stuff to do"

### Tip 3: Use Descriptions
Fill in the project description with the original goal. This helps you remember why you created the project.

### Tip 4: Keep Due Dates Realistic
Don't set due dates in the past or too soon. Give yourself realistic time to complete all tasks.

### Tip 5: Regular Reviews
Every week, check your active projects:
- Update task progress
- Adjust due dates if needed
- Add new tasks if scope changed
- Archive completed projects
